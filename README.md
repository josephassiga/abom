# abom.ai - website

This repository holds the public **abom.ai** website only. It is served via
GitHub Pages from [`website/`](website/) (see
[`.github/workflows/pages.yml`](.github/workflows/pages.yml)).

The ABOM project itself — the specification, the reference CLI, examples,
strategy and architecture docs — lives in a separate private repository.

- **Website:** <https://abom.ai>
- **License:** [Apache-2.0](LICENSE)

The site (`website/index.html`) is fully self-contained: all images are inlined,
with no build step.
