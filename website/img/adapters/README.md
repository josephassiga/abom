# Adapter logos

Drop each provider's logo here with the **exact filename** below. The adapter
catalog (`adapters.html`) renders these as the tile icon; if a file is missing,
the tile falls back to a letter glyph (so nothing ever shows a broken image).

Recommended: square-ish **SVG** (or transparent PNG), rendered at 22×22 inside a
34px rounded badge. Keep them simple — they appear small.

| OpenAI         | `openai.svg`          |
| Mistral        | `mistral.svg`         |
| Google         | `google.svg`          |

| Groq           | `groq.svg`            |

| Ollama         | `ollama.svg`          |
| vLLM           | `vllm.svg`            |
| Hugging Face   | `huggingface.svg`     |

If you use PNGs instead of SVGs, just change the extension in the filename and
update the matching `img:` entry in the `ADAPTERS` array in `adapters.html`.

**Note on trademarks:** these are third-party brand logos. Use the official
press/brand assets and follow each provider's brand guidelines.
